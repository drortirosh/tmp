GSNv2 Test App

This page can be used to test support of browsers for GSN methods. the log is pushed (in reverse) into the page, so we can see the log even of a mobile browser with no debug console. To check signature, add URL params (need to refresh page to take effect)
  • add _v3 suffix, stringify=false
  • add _v3 suffix, stringify=true
  • add _v4 suffix, stringify=false
  • add _v4 suffix, stringify=true
